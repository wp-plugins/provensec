Simple plugin for automating registration and ecommerce process for Cloudpro-X security scanning using provensec API.

== Description ==
NO CODING required. Simple but effective way to use signup to Provensec services via API using this plugin.

It's used to provide sign up form to automate registeration via API for Scanning services.Users register , make payment and are able to access cloudprox dashboard where they can add websites and IP addresses to enable security scanning.

By default plugin creates a sample page for signup with shortcode. Either You can use this page or simply put the shortcode "[provensec-registration-form]" to display signup form on page or post.

Easy steps to Go:

* Get API (25 characters) key from your account setting. You need to signup with www.provensec.com for this.
* Add your registered username for verification. 
* Provide max. number of assets and price per asset.
* Enter your business paypal email address and logo of your site.
* Select the payment success page.
* You can enable captcha to stop spam.

Now you are ready to use provensec signup service. You can view all registered users from WP admin backend.

== Installation ==

1. Upload `provensec.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the shortcode `[provensec-registration-form]` in your page or post
